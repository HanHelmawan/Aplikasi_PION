# Pion

Aplikasi Jasa Hiperlokal & Pendampingan Cepat

## Deskripsi
Pion adalah sebuah platform yang menghubungkan pencari jasa dengan penyedia jasa secara hiperlokal (berbasis lokasi terdekat).

Lihat [SDD.md](SDD.md) untuk detail arsitektur dan spesifikasi desain fitur seperti:
- KYC & Verifikasi Identitas
- Escrow Wallet (Sistem Pembayaran Aman)
- Tombol Darurat (SOS)
- Geofencing & Live Tracking

## Setup untuk Tim
1. Clone repositori ini.
2. Buat file `mcp.json` di root folder dengan referensi dari tim (berisi API keys).
3. Jalankan `flutter pub get`.
4. Jalankan aplikasi di emulator/device dengan `flutter run`.
